# Attention Engine 4.0

An interactive web experiment built for Web Engineering Lab 6.

## 🚀 Concept
This project explores how interactive UI design, gamification, and controlled visual chaos can increase user attention span on a webpage.

Inspired by:
- https://arngren.net/
- https://wwwwwwwww.jodi.org/

## 🧠 Features
- 2-Second Attention Scan
- Auto Glitch Effects
- Scroll Reveal Animations
- Mini-Game (Catch the Glow)
- Web Audio API Integration
- 3D Parallax Interaction
- Matrix Visual Effect
- Form Validation (Custom JavaScript)
- Glassmorphism UI Design

## 🛠 Technologies Used
- HTML5
- CSS3 (Animations, Glass UI)
- JavaScript (ES6)
- Web Audio API
- Intersection Observer API
